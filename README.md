## My Projects

### [Windows 9x QuickInstall](https://github.com/oerg866/win98-quickinstall)

A framework + installer to (very) quickly install Windows 98 on anything from a 486 up to a modern system

### [ethersrv-866](https://github.com/oerg866/ethersrv-866)

An improved fork of Mateusz Viste's EtherDFS server with added case-insensitive and long file-name support (making it useful on non-FAT32 file systems) + Mac OS X and FreeBSD support

### [K6INIT](https://github.com/oerg866/k6init)

AMD K6-2 (CXT) / K6-2+ / K6-3 / K6-3+ Write Allocate / Write Combine / Write Ordering / Frequency Multiplier / Cache Enable Initialization driver for MS-DOS

### [VIA_AC97.866](https://github.com/oerg866/via_ac97.866)

DOS Initialization Driver & Mixer for VIA VT82C686A/B AC'97 Audio SoundBlaster Pro mode + OPL3 FM Emulation TSR

### [em8300-866](https://github.com/oerg866/em8300-866)

Updated fork of the em8300 / REALmagic Hollywood Plus / Creative DXR3 drivers for modern Linux kernels

### [lib866d](https://github.com/oerg866/lib866d)

Library for writing real-mode DOS software with Microsoft C 6.x-8.x, Borland Turbo C and OpenWatcom

### [anbui](https://github.com/oerg866/anbui)

(Very) Tiny Text UI library written in C (DOS, Win32, Linux compatibility so far)

### [TRW-BiosMods](https://github.com/oerg866/TRW-BiosMods)

Retro motherboard BIOS modifications released on TheRetroWeb

### [pciedit](https://github.com/oerg866/pciedit)

Program to edit PCI Configuration Registers for MS-DOS and compatibles (e.g. FreeDOS)
